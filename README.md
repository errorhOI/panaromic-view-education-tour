# Panoramic View Education Tour

## Overview
This project is a web-based panoramic education tour system that allows users to explore a virtual environment through interactive 360-degree panoramic views. The application is designed to provide an engaging and visually immersive way for users to navigate educational or tour-based spaces.

The project uses panorama rendering libraries together with front-end web technologies to display panoramic scenes and support user interaction.

## Features
- 360-degree panoramic environment viewing
- Interactive tour-style experience
- Web-based interface
- User-friendly navigation
- Visual educational presentation

## Technologies Used
- HTML
- CSS
- JavaScript
- Three.js
- Pannellum / Panolens-style panoramic library

## Project Files
- `index.html` – Main webpage structure
- `style.css` – Styling for the interface
- `main.js` – Main JavaScript logic
- `three.min.js` – Three.js library
- `panolens.min.js` – Panoramic viewer library
- `Images/` – Image assets used in the project
- `Recording 2026-04-16 140126.mp4` – Demo video of the project

## Project Purpose
The purpose of this project is to create an interactive panoramic-view education tour that allows users to explore locations visually in a more immersive and engaging way compared to static images or traditional page-based content.
